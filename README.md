# Admission Predictor (Streamlit)

Streamlit app to train and predict graduate admission chances.
- Supports upload of `Admission_Predict.csv` (or commit the CSV to repo root).
- Trains sklearn models (Linear Regression, Decision Tree, Random Forest).
- No TensorFlow required.

1. Clone repo:
```bash
git clone https://github.com/<your-username>/admission-streamlit.git
